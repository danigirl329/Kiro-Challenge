Overview

This is a RESTful API for managing events, built with FastAPI and deployed on AWS Lambda with DynamoDB as the data store.

Base URL: https://myto1dwcaa.execute-api.us-west-2.amazonaws.com/prod/

Modules

main.py

Main FastAPI application with all API endpoints and middleware configuration.

create_event(event: EventCreate)

Create a new event in the database.

event: EventCreate - Event data to create
Returns: Event (201 Created)
get_all_events(status: str = None)

Retrieve all events, optionally filtered by status.

status: str (optional) - Filter events by status
Returns: List[Event] (200 OK)
get_event(event_id: str)

Retrieve a specific event by ID.

event_id: str - Unique event identifier
Returns: Event (200 OK) or 404 Not Found
update_event(event_id: str, event_update: EventUpdate)

Update an existing event.

event_id: str - Event ID to update
event_update: EventUpdate - Fields to update
Returns: Event (200 OK) or 404 Not Found
delete_event(event_id: str)

Delete an event from the database.

event_id: str - Event ID to delete
Returns: 204 No Content or 404 Not Found

models.py

Pydantic models for request/response validation.

Event

Complete event model with all fields.

eventId: str - Unique identifier
title: str (1-200 chars) - Event title
description: str (1-1000 chars) - Event description
date: str - Event date (ISO 8601 format)
location: str (1-200 chars) - Event location
capacity: int (1-100000) - Maximum attendees
organizer: str (1-100 chars) - Event organizer
status: str - Event status (draft|published|cancelled|completed|active)
EventCreate

Model for creating new events. EventId is optional and auto-generated if not provided.

EventUpdate

Model for updating events. All fields are optional.

database.py

DynamoDB operations for event management.

create_event(event_data: dict)

Insert a new event into DynamoDB.

event_data: dict - Event data to insert
Returns: dict - Created event with generated eventId
get_event(event_id: str)

Retrieve a single event by ID.

event_id: str - Event identifier
Returns: Optional[dict] - Event data or None
get_all_events()

Scan and retrieve all events from DynamoDB.

Returns: List[dict] - List of all events
update_event(event_id: str, update_data: dict)

Update specific fields of an event.

event_id: str - Event to update
update_data: dict - Fields to update
Returns: Optional[dict] - Updated event or None
delete_event(event_id: str)

Remove an event from DynamoDB.

event_id: str - Event to delete
Returns: bool - Success status

API Endpoints

POST /events

Create a new event

{
  "eventId": "optional-custom-id",
  "title": "Event Title",
  "description": "Event description",
  "date": "2024-12-15",
  "location": "Event Location",
  "capacity": 100,
  "organizer": "Organizer Name",
  "status": "published"
}
GET /events

Get all events (supports ?status=active query parameter)

GET /events/{event_id}

Get a specific event by ID

PUT /events/{event_id}

Update an event (partial updates supported)

  "capacity": 150

Delete an event
